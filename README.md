# deb-repos


    Angular2
    Webpack2

Basic Usage

# clone the repo
  git clone https://github.com/nodesha88/deb-repos.git
  cd deb-repos

# install 
  npm install
  typings install

# build for development/production
  npm run build:dev
  npm run build:prod

# run development/production
  npm run dev
  npm start
